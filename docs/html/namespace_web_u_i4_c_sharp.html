◆ webui_config

enum WebUI4CSharp.webui_config

WebUI configuration.

Enumerator
show_wait_connection 

Control if webui_show(), webui_show_browser() and webui_show_wv() should wait for the window to connect before returns or not. Default: True.

ui_event_blocking 

Control if WebUI should block and process the UI events one a time in a single thread True, or process every event in a new non-blocking thread False. This updates all windows. You can use webui_set_event_blocking() for a specific single window update. Default: False.
